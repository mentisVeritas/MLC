# MLC
ML Community — lessons, homework, contests, shared `data/`.

## Layout
- **Lessons/** — numbered topics: statistics & viz, data wrangling, classical ML, deep learning, NLP, SQL.
- **Homeworks/notebooks/** — assignment Jupyter notebooks (paths use `../../data/`).
- **Homeworks/sql/** — SQL homework scripts.
- **Contests/** — competition notebooks and local CSVs.
- **projects/** — standalone mini-projects (e.g. letter recognition).
- **data/** — datasets used across lessons and homework.
